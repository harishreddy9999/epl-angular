EPL website

This page contains my final project of Level 4, which was to create a responsive SPA to display the match stats of EPL for year 2015-16 and 2016-17, it contains a list view of all the matches whose data is provided to me in JSON format, it has following views :

  • A view to display all matches held. This view contains various filters like filter by year, filter by Team name, filter by score etc.
  • A view to display details of a single match.
  • A view to display team wise statistics like total matches played, total won, total lost, total number of goals scored by that team etc.
